### Installation && Compile Instructions

Open your Command Line Interface (CLI) and run these commands. Once done go to
http://localhost:8080/

First Tab:
``` bash
$ cd Desktop/
$ git clone https://github.com/scvGeekText/GeekBookStore.git
$ cd GeekBookStore/
$ npm install
$ npm run dev
```

Second Tab:
``` bash
npm start
```
